# Tutorial System

## Responsibility
- track tutorial step
- send step updates to client
- gate early-game onboarding

## Steps
- INFLATE_FIRST
- INFLATE_SECOND
- BUY_BALLOON
- EQUIP_BALLOON
- INFLATE_NEW
- GET_PET
- PLACE_STAND
- COLLECT_INCOME
- BUY_PINK
- COMPLETED

## Rules
- tutorial step is server-owned
- client only shows UI and reacts to step changes
