# Поставки шаров в магазин (ARPDAU)
- В магазине продажа шаров ограничено. Каждые 15 минут поступают новые шары в продажу

# Нить от шара
Между шаром и игроком находится гибкая нить состоящая из звеньев. Одна сторона нити притягивается к игроку когда он подбегает к шару (и отвязывается когда игрок отбегает от шара). Другая сторона прикреплена к шару и поднимается за ним, когда тот взлетает.

# Золотой шар с эффектом перетягивания (Progress Loop)
Из этого шара можно выбить питомца только из коллекции Gold.

# Алмазный шар (Progress Loop)
- Из этого шара можно выбить питомца из коллекции Diamond

# Космический шар(ARPPU)
Донатный шар из него выпадают только Легкндарные предметы из всех возможных коллекций. RaritySystem исключает возможно выпадения бомбы

# Гонка рейтинга (Event)
За сжатый срок игрок должен набрать больше рейтинга чем остальные
- Используется уникальный шар
- Очки зарабатываются с этого шара

# Index (+ Retention)
Отображает всех питомцев которые игрок когда-либо имел

# Rebirth Save (+ ARPDAU)
Игрок может сохранить определенное количество шаров после Rebirth

# Система лояльности (+ Session Length, Retention)
Предотвращает вероятность фрустрации игрока когда ему долгое время не выпадает ценный предмет

# Система заданий
Игрок последовательно получает заданий и за их выполнения зарабатывает золотые ключи

# Система рейтинга
- Игрок собирает комбинацию из удачных надуваний шара, получает за это определённый ранг.
- Ранг увеличивается постепенно, не перепрыгивая через один
- Ранг игрока отображается с правой стороны от ника игрока
- Ранг зависит от того, какой шар надувался
- Чем больше шар игрок надувает, тем больше очков прибавляется к рангу
- Если шар лопается, то очки убавляются
- Названия рангов: Common, Uncommon и тд.

# Перетягивание питомца
С определённой долей вероятности рулетка запустится с двумя питомцами, которая в последний момент запустит интерактив, где игроку придётся нажимать правую или левую кнопку мыши, чтобы выбить именно того питомца который он захочет
